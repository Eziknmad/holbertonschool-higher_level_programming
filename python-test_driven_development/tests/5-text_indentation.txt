The ``5-text_indentation`` module
==================================

Using ``text_indentation``
--------------------------

Import the function:
    >>> text_indentation = __import__('5-text_indentation').text_indentation

Test with simple sentence ending with period:
    >>> text_indentation("Hello.")
    Hello.
    <BLANKLINE>

Test with question mark:
    >>> text_indentation("How are you?")
    How are you?
    <BLANKLINE>

Test with colon:
    >>> text_indentation("Here:")
    Here:
    <BLANKLINE>

Test with multiple delimiters:
    >>> text_indentation("Hello. How are you? Fine: thanks.")
    Hello.
    <BLANKLINE>
    How are you?
    <BLANKLINE>
    Fine:
    <BLANKLINE>
    thanks.
    <BLANKLINE>

Test with spaces after delimiters:
    >>> text_indentation("Hello.    How are you?")
    Hello.
    <BLANKLINE>
    How are you?
    <BLANKLINE>
