# JavaScript DOM Manipulation

This project contains exercises for learning JavaScript DOM manipulation techniques.

## Description
This project focuses on understanding how to manipulate the DOM using JavaScript, including selecting elements, modifying styles, updating content, and making network requests without reloading the page.

## Learning Objectives
- How to select HTML elements in JavaScript
- Differences between ID, class and tag name selectors
- How to modify an HTML element style
- How to get and update an HTML element content
- How to modify the DOM
- How to make requests with XmlHTTPRequest and Fetch API
- How to listen/bind to DOM and user events

## Requirements
- All files interpreted on Chrome browser (version 57.0 or later)
- Code is semistandard compliant
- No use of `var` keyword
- HTML should not reload for each action

## Files
- `0-script.js` - Script that updates the text color of the header element to red
- `0-main.html` - HTML file for testing the script
